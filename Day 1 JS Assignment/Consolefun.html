Various method of Console function

1) warn() function is used to print warning messages to the console output with warning symbol

2) info() function displayed Informational messages. The info() function will provide information symbol to the console output.

3) error(): We can print the error messages to the console with the error()function. An error symbol will be displayed in the outputs and the output text color will be red to express this is an error.

4) log(): The most popular function of the console object is log() function which will simply print given data, text or object into the console output.

5) trace: While running JavaScript code on the browser DOM there will be some stack about running JavaScript code. We can print this stack to the console output with the trace() function for the currently executed code.

6) table() function is used to display given data or object in a table formatted way. The object or data will be generally in JSON format.

7) group(): We can use groups in order to print messages in hierarchical manner. Groups can group() and end with groupEnd() functions. Groups can be nested as inner group list a regular group.
